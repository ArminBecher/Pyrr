Utilities¶

Provides common utility functions.

pyrr.utils.all_parameters_as_numpy_arrays(fn)[source]¶

Converts all of a function’s arguments to numpy arrays.

pyrr.utils.parameters_as_numpy_arrays(*args_to_convert)[source]¶

Converts specific arguments to numpy arrays.

Arguments are specified by their argument name. Ie:

@parameters_as_numpy_arrays( 'a', 'b', 'optional' )
def myfunc( a, b, *args, **kwargs ):
    pass

myfunc( 1, [2,2], optional = [3,3,3] )
