Sphere¶

Provide functions for the creation and manipulation of 3D Spheres.

A sphere is defined as a position (3d vector) and a radius (float).

pyrr.sphere.create_from_points(*args, **kwargs)[source]¶

Creates a sphere centred around 0,0,0 that encompasses the furthest point in the provided list.

@return: Returns a sphere as a two value tuple. The first value is the sphere’s position. The second value is the sphere’s radius.
